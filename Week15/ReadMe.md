## 03 并发编程
### 为了保证线程安全，使用锁，原子类等机制来满足需求，同时在使用锁时需要注意的点是锁的范围尽量要小，细分锁的粒度。为了能够服用创建的线程，可以使用线程池来创建，异步结果的读取与异步编排通过CompletableFuture实现。
